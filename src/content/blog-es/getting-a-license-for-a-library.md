Si quieres la versión completa de una de nuestras bibliotecas especiales, necesitas comprar una licencia. Esta licencia estará asociada solo a tu computador.

**Paso 1 - Obtener tu ID de computador**

Descarga la siguiente aplicación (construida en AnyLogic) para obtener tu ID de computador. Asegúrate de ejecutar esto en el computador donde vas a usar AnyLogic:

[https://www.dropbox.com/scl/fo/sm018dsyqek0ezcbqvh4k/AL3t7mAnUa7c4F9WGGBmLF8?rlkey=grapj1hbydvtviohxi51a6n4a&dl=0](https://www.dropbox.com/scl/fo/sm018dsyqek0ezcbqvh4k/AL3t7mAnUa7c4F9WGGBmLF8?rlkey=grapj1hbydvtviohxi51a6n4a&dl=0)

Ten en cuenta que se necesita una conexión a internet para usar este software. Cuando ejecutes el archivo ejecutable, verás la siguiente interfaz que te mostrará cuál es tu ID de computador único, y podrás hacer clic en el botón para copiar ese ID (así no tienes que escribirlo manualmente)

![](/images/blog/getting-a-license-for-a-library/Monosnap-License-_-Simulation-AnyLogic-Professio.png)

**Paso 2 - Pagar y enviarnos tu ID**

Ahora tienes tu ID único, ahora es momento de pagar. Por ahora solo aceptamos pagos usando PayPal, y puedes crear fácilmente una cuenta y colocar la información de tu tarjeta de crédito para hacer pagos.

Expandiremos las opciones y actualizaremos la información en esta página. Para hacer el pago, puedes usar el siguiente enlace:
https://www.paypal.com/paypalme/noorjax

Si no conoces el precio de una biblioteca, por favor envíanos un correo electrónico a info@noorjax.com para obtener la información. Dinos qué biblioteca quieres, tu país de residencia y si eres estudiante o profesional.

En la página de PayPal tendrás que indicar una razón para el pago, por favor incluye allí tu correo electrónico y la biblioteca que quieres usar. Y asegúrate de que ese correo electrónico sea el mismo que estás usando para comunicarte con nosotros. Este es el correo electrónico que estará asociado a tu licencia.

![](/images/blog/getting-a-license-for-a-library/paypal-1.png)

**Paso 3 - Verificar tu compra**

Una vez que hayas pagado, crearemos un token para ti y te enviaremos un correo electrónico de confirmación usando la información que proporcionaste en el campo "what's this for" del pago en PayPal. Si no recibes un correo electrónico en 24 horas, contáctanos porque es muy probable que no hayas escrito la información correcta y por eso no pudimos contactarte.
Una vez que recibas nuestro correo electrónico confirmando la compra, puedes verificar tu token usando la misma aplicación que usaste para obtener tu ID. Verás una lista de las licencias activas que tienes y obtendrás tu token que usarás para activar tu licencia para la biblioteca que quieres usar:

![](/images/blog/getting-a-license-for-a-library/Monosnap-Video-Preview-2024-09-09-20.45.48.png)

![](/images/blog/getting-a-license-for-a-library/Monosnap-Video-Preview-2024-09-09-20.46.11.png)

**Paso 4 - Activar tu biblioteca**

Todas las bibliotecas tienen un objeto de configuración que necesitas usar. El token que obtienes se usará en la configuración de esa biblioteca como entrada. Puedes leer más en la documentación de la biblioteca. Si tienes problemas, no dudes en enviarnos un correo electrónico.
