## Descripcion del Curso

Esta sesion demuestra las mejores practicas para exportar datos desde modelos de simulacion. Los participantes aprenderan a definir parametros usando clases personalizadas, aprovechar la libreria Lombok para un codigo mas limpio e implementar flujos de trabajo profesionales de exportacion a archivos de texto, Excel y bases de datos.

---

## Objetivos

- Aprender las mejores practicas para la definicion de parametros usando clases personalizadas
- Comprender las ventajas de la libreria Lombok en AnyLogic y como usarla
- Aprender a recopilar y guardar datos de forma sistematica dentro de un modelo
- Dominar las tecnicas de exportacion de datos para modelos de simulacion en AnyLogic

---

## Prerrequisitos

**AnyLogic**
- Familiaridad con la interfaz, View Areas, Agentes, Eventos, Funciones, Bases de datos

**JAVA**
- Variables, Parametros, Casting

**Eventos Discretos**
- Source, Resources, Service, Seize, Release, MoveTo, Transporters, Delay, Sink

**Fundamentos Estadisticos**
- Distribucion uniforme, distribucion triangular

---

## Temas Cubiertos

### 1. Definicion de Parametros

- Agregar parametros usando una clase Java personalizada
- Lectura de valores de parametros desde Excel en tiempo de ejecucion

### 2. La Libreria Lombok

- Que es la libreria Lombok y sus ventajas
- Como configurar y usar Lombok en proyectos de AnyLogic

### 3. Exportacion de Datos desde Experimentos de Simulacion

- Agregar una clase para cada tabla de exportacion
- Guardar instancias de clases en colecciones — la clase general de exportacion
- Exportacion a archivos de texto
- Exportacion a archivos Excel
- Exportacion a una base de datos

---

## Conocimientos Adquiridos

Al finalizar esta sesion, los participantes seran capaces de:

- Definir parametros en una clase personalizada y leer valores desde Excel
- Configurar la libreria Lombok en proyectos de AnyLogic
- Implementar la recopilacion estructurada de datos dentro de modelos de simulacion
- Exportar datos de simulacion a archivos de texto, Excel o bases de datos usando patrones profesionales
